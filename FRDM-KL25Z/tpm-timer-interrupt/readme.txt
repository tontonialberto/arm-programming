Author:
    Alberto Tontoni

Description:
    This program blinks the built-in red and blue LEDs at different frequencies
	using TPM0 and TPM1 in timer mode, making use of interrupts.

Requirements:
    -Freescale FRDM-KL25Z development board with CMSIS-DAP debug/programming app.
    -No external hardware is required.

Credits:
    Chen et al. - "Freescale ARM Cortex-M Embedded Programming Using C Language"
